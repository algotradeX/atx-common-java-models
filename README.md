#### Install 1/2: Add this to pom.xml:

```xml
<dependency>
  <groupId>com.atx.java</groupId>
  <artifactId>atx-common-java-models</artifactId>
  <version>0.0.1</version>
</dependency>
```

#### Install 2/2: Run via command line

```bash
$ mvn install 
```
